Three.js Classic

About This Project

This 3D Web Application was built for the 3D Web Applications module. It allows users to view, interact with, and swap between multiple 3D models entirely in the browser using Three.js.

Features

  • Interactive 3D viewer with OrbitControls
  • Multiple local/remote GLTF & GLB models
  • Camera presets and wireframe mode
  • Progress overlay while loading

Accessibility

Buttons have clear labels, and models can be paused to allow static inspection. The UI uses Bootstrap for consistent styling and responsive design. Keyboard navigation is supported for dropdown and buttons.

Architecture

The app follows a simple client-side architecture using HTML, Bootstrap, and Three.js. The 3D rendering runs in the main browser thread with no backend required. Models are loaded via GLTFLoader from local /models/ folders or from remote URLs as a fallback.
